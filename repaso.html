Recuerda utilizar los selectores combinados o selectores de descendencia directa para que se haga más fácil aplicar los estilos. Ejemplo:


      thead>tr>th { /* todos los th dentro de un tr dentro de un thead */
        border-radius: 15px;
        color: white;
        padding: 5px;
      }
