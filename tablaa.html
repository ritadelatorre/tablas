Para hacer el efecto pijama en la tabla de arriba, se coloca el siguiente CSS:


      tr:nth-child(even) {
        background-color: #f2f2f2; /* Color gris claro para filas pares */
      }

      tr:nth-child(odd) {
        background-color: white; /* Color blanco para filas impares */
      }

Usamos las pseudo-clases nth-child(even) para seleccionar las filas pares (2, 4, 6...) y nth-child(odd) para seleccionar las filas impares (1, 3, 5...).
